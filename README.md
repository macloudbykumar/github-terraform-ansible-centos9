# GitHub + Terraform + Ansible Automation on CentOS 9

## Overview

This repository demonstrates how to automate infrastructure provisioning and server configuration using **Terraform**, **Ansible**, and **GitHub** on **CentOS Stream 9**.

The project is designed for DevOps Engineers, Cloud Engineers, Platform Engineers, System Administrators, and students who want hands-on experience with Infrastructure as Code (IaC), Configuration Management, and CI/CD automation.

By combining Terraform and Ansible, organizations can provision infrastructure and configure operating systems consistently, securely, and at scale.
